# Career Compass App

A beautiful, thoughtful career planning application designed to help professionals define their North Star vision and create actionable career paths.

## ğŸ§­ Overview

The Career Compass app guides users through a structured 4-step framework to move from long-term vision (5-10 years) to immediate action (0-90 days). Unlike rigid career planning tools, this compass-based approach adapts as your goals and circumstances evolve.

## âœ¨ Features

- **Beautiful Landing Page**: Introduces the compass metaphor and highlights the 4 key sections
- **4-Step Guided Journey**:
  1. **North Star Vision** (5-10 years): Define your ideal future state and purpose
  2. **Cardinal Directions**: Map your values, strengths, energizers, and gaps
  3. **Map the Terrain** (3-5 years): Identify milestones, constraints, and opportunities
  4. **Start the Route** (0-90 days): Create immediate actions and foundational habits
- **Dynamic Summary Page**: Synthesizes all inputs into a comprehensive career compass
- **Print/Export Functionality**: Save your compass as PDF or text file
- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices
- **Modern UI**: Built with React, Tailwind CSS, and Lucide icons

1. **Start on the Landing Page**: Review the framework and understand the compass metaphor
